## 📌 Project Overview
An end-to-end Machine Learning system that predicts a patient's 
risk of diabetes based on clinical health measurements.

Trained on the real **Pima Indians Diabetes Dataset** from the 
National Institute of Diabetes and Digestive and Kidney Diseases.

---

## 🎯 Features
- ✅ Complete Exploratory Data Analysis (EDA)
- ✅ Smart Missing Value Imputation
- ✅ Feature Engineering (4 new clinical features)
- ✅ 5 ML Models trained and compared
- ✅ Hyperparameter Tuning with GridSearchCV
- ✅ ROC Curve and Confusion Matrix Evaluation
- ✅ Live Web App deployed with Streamlit

---

## 🧠 Models Used
| Model | Type |
|---|---|
| Logistic Regression | Baseline |
| K-Nearest Neighbors | Distance-based |
| Decision Tree | Tree-based |
| Random Forest | Ensemble ✅ Winner |
| Gradient Boosting | Ensemble |

## 📊 Dataset
- **Source:** Pima Indians Diabetes Database — UCI / Kaggle
- **Patients:** 768 female patients
- **Features:** 8 clinical measurements
- **Target:** Diabetes Yes/No
